4.1 Description

Skills are the abilities an Actor can employ in battle. Skills are stronger than regular weapon attacks, but have certain conditions that need to be met in order to use them.

4.2 Nature

Skills are of two different natures: Physical and Magic.

4.2.1 Physical

Physical Skills draw from the attributes of ATK and DEF. Using a Physical Skill consumes a set amount of MP. Physical Skills can be used instantaneously, but require a short cooldown period before they can be used again. Physical Skills are geared towards gaining a high CC as they can be easily chained together. Physical Skills can be blocked or avoided, thus ending the CC.

4.2.2 Magic

Magic Skills draw from the attributes of MAG and RES. Using a Magic Skill consumes a proportionate amount of MP depending on how long you charge the spell. A spell typically has three stages: Weak, Regular, Strong. Using the same skill on weak mode is very different from using it on strong mode. Magic Skills are geared towards exploiting elemental weakness which slows down the enemy’s Overdrive gauge. Magic Skills can’t be blocked or avoided, but can be absorbed based on the enemy’s elemental affinity.

4.3 Types

Skills are further divided into the following categories: Dynamic, Passive, Auto and Junction.

4.3.1 Dynamic Skills

Dynamic Skills are abilities used by Actors in battle (e.g “Tsubame Gaeshi” lets the Actor perform three consecutive slashes upon an Enemy). Up to four Dynamic Skills can be mapped to a set of buttons of an Actor for any given battle. These skill-button mappings can be changed only in the Actor’s Status Screen between battles.

4.3.2 Passive Skills

Passive Skills are abilities that enhance an Actor’s attributes, resistances, immunities or performance (e.g. “Anti-Venom” improves Actor’s chances of not getting poisoned by 20%). Up to three Passive Skills can be assigned to an Actor for any given battle. These skills can be changed only in the Actor’s Status Screen between battles.

4.3.3 Auto Skills

Auto Skills are abilities that will automatically kick-in under certain conditions (e.g. “Riestrad” causes an Actor to obtain the “Bloodlust” Status Effect once 50% HP is depleted). Up to three Auto Skills can be assigned to an Actor for any given battle. These skills can be changed only in the Actor’s Status Screen between battles.

4.3.4 Junction Skills

Junction Skills are special abilities that are used by the entire party on a specified target or targets. Junction Skills cost no MP, but can only be used when the Overdrive gauge has reached the required tier. Once used, the Overdrive Gauge will drop by the number of tiers consumed. The Junction Skill performed depends on the people in the Party.
